# PocketCMS デモ

PocketCMS の一次仕様（モバイルファースト／固定CTA前提）を再現した簡易デモです。ブラウザのみで動作し、ローカルストレージに保存した内容が公開ページに即時反映されます。

## 構成
- `public/index.html`: 公開サイトのテンプレート。固定フッターCTA、メニュー、施術例、FAQ、分析パネルを表示します。
- `public/admin.html`: 管理画面デモ。スマホ更新を想定したタブ構成で、サイト情報・CTA・投稿・FAQを編集できます。
- `public/data/sample-site.json`: 初期データ。管理画面で編集した内容はブラウザのローカルストレージに保存されます。
- `public/assets/placeholder.svg`: 画像の簡易プレースホルダー。

## 使い方
1. 任意の静的サーバーで `public` ディレクトリを配信します。例: `python -m http.server 8000 --directory public`
2. 管理画面: `http://localhost:8000/admin.html` を開き、内容を編集して保存します。
3. 公開サイト: `http://localhost:8000/index.html` を更新すると、保存した内容が反映されます（ローカルストレージ経由）。

## 特徴
- 固定フッターCTA（予約 / LINE / 電話）と「本日の空き」強調表示。
- 施術例投稿は複数画像・タグ・CTA・実績メモをサポート。
- QR / 短縮URL領域とOGP想定のメタ情報（導線ツール）。
- 予約・LINE・電話のタップ数を想定した簡易分析カード。

## 備考
- バックエンドや永続化は含まれていません。MVPのUI体験と情報設計にフォーカスしたデモです。
- 仕様の詳細は `PocketCMS仕様書.md` を参照してください。
